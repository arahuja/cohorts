Cohorts
=======

Cohorts is a library for analyzing clinical data, mutations and neoepitopes in patient cohorts.

Installation
------------

You can install Cohorts using [pip](https://pip.pypa.io/en/latest/quickstart.html):

```bash
pip install cohorts
```
